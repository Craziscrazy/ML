1. Rain Prediction using Logistic Regression
Overview
Rain prediction is a binary classification problem where the goal is to predict whether rainfall will occur based on weather-related parameters.
Logistic Regression is a supervised machine learning algorithm widely used for classification problems.

Objective
To predict rainfall occurrence using historical weather data.

Features
Typical input features include:
Temperature
Humidity
Wind Speed
Atmospheric Pressure
Cloud Cover
Previous Rainfall Data

Methodology
The following steps were followed:
Data collection and loading
Data preprocessing
Feature selection
Train-test split
Model training using Logistic Regression
Prediction and performance evaluation

Evaluation Metrics
Accuracy
Confusion Matrix

Technologies Used
Python
Pandas
NumPy
Scikit-learn
Matplotlib
Plotly
